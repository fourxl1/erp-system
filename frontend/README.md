# ERP Frontend

This frontend is the React/Vite client for the ERP inventory, movement, and request workflow.

## Run

```powershell
cd C:\Users\Prince\Desktop\erp_system\frontend
npm run dev
```

Default dev URL: `http://localhost:5173`

## API Configuration

The frontend auto-detects the backend using the current browser hostname and port `5000`.
Set `VITE_API_BASE_URL` only when you need to override that default.

## Key Screens

- Dashboard
- Inventory
- Stock Movements
- Requests
- Messages
- Reports
- Maintenance
- Master Data
